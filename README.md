To run this application you will need Visual Studio, .NET 9.0 and .NET MAUI installed.
Clone the repository then build the application. The application is designed primarily for Android so you will need to connect an Android device and run there, or use the .NET MAUI emulator.
In order to see leaderboards for different months, progress, etc. you will need to acess an account that has existed for a number of months. I have created an example account for organisation and region.
